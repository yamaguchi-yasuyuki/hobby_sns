# UI/UX方向性

## 方針

### 1. シンプルで直感的な操作
- **認知負荷の軽減**: 1画面あたりの情報量を絞り、ユーザーが迷わない導線を設計する
- **操作ステップの最小化**: 目的達成までに必要なクリック・タップ数を減らし、3ステップ以内を目安とする
- **一貫したパターン**: ナビゲーション、ボタン配置、フォームの流れなど、画面間で統一された操作パターンを採用する
- **明確なフィードバック**: 操作結果（成功・失敗・処理中）を即座に視覚的に伝える

### 2. 余白を活かした読みやすいレイアウト
- **適切な余白**: 要素間のスペーシングは最低でも8px単位（8, 16, 24, 32px）で統一し、窮屈さを避ける
- **視覚的階層**: 見出し・本文・補足情報の区別を明確にし、視線の流れを自然に誘導する
- **行間・字間**: 長文は行間1.5〜1.75、適度な字間を確保して可読性を高める
- **コントラスト**: 背景と文字のコントラスト比を確保し（WCAG AA基準: 4.5:1以上）、読みやすさを担保する

### 3. モバイルでも使いやすいタッチターゲット
- **最小タッチサイズ**: ボタン・リンク・アイコンは最低44×44px（推奨48×48px）を確保する
- **タップ領域の拡張**: 見た目より小さく見える要素は、ヒット領域を視覚より広く取る
- **要素間の間隔**: タッチターゲット同士は最低8px以上の間隔を空け、誤タップを防ぐ
- **片手操作を考慮**: 重要なアクションは画面下部〜中央に配置し、親指で届きやすい位置を意識する
