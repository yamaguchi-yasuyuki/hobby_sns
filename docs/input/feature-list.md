## 機能詳細

### 「気になる」保存
**概要：** コンテンツをコレクションに1タップで追加する

**ユーザーストーリー：**
「40代の趣味人として、気になったキャンプ場や古着を後で見返したい。なぜなら、その場でアクションできなくても、週末の計画時に参考にしたいから。」

**入力：**
- コンテンツカードの「気になる」ボタンタップ
- カテゴリ選択（省略可。デフォルトカテゴリに保存）

**出力：**
- コレクションに追加完了のトースト通知
- 「気になる」ボタンが保存済み状態に変化

**受け入れ条件：**
- [ ] ログイン状態で「気になる」をタップするとコレクションに追加される
- [ ] 未ログインでタップした場合、ログイン/登録を促すモーダルが表示される
- [ ] 同じコンテンツを再度タップすると「気になる」が解除される
- [ ] コレクション画面で保存したコンテンツが確認できる
