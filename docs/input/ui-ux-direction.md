# UI/UX方針

## 1. 全体の方向性

### デザインテーマ
**「静かな上質感」** ── 落ち着いていて、眺めていて飽きない。騒がしくない。

40代男性が就寝前や移動中にリラックスして使えるデザイン。キャンプ・温泉・バイク・釣り・古着・ウイスキーなどのライフスタイル系コンテンツが映えるビジュアル体験を重視しつつ、SNS特有の「数字（いいね・フォロワー）の圧迫感」を完全に排除する。

### キーワード
- 静寂（Quiet）
- 上質（Quality）
- 没入（Immersive）
- シンプル（Simple）
- 余白（Breathing room）

### 参考サービス
| サービス名 | 参考にする点 |
|------------|--------------|
| Instagram | ビジュアル中心のフィード体験、カード型のレイアウト |
| Pinterest | 投稿しなくてもコレクションが楽しい体験 |
| Monocle（雑誌） | 落ち着いた上質感のある世界観 |
| Apple Music | 静かで洗練されたUIの雰囲気 |

## 2. ビジュアル

### カラー方針
- メインカラー：ディープグリーン（#2D4A3E）またはチャコールグレー（#3A3A3A）
- アクセントカラー：アースブラウン（#8B6F47）または砂色（#C4A882）
- 背景：オフホワイト（#F7F5F2）またはダークモード時にディープネイビー（#1A1F2E）
- テキスト：ダークグレー（#2C2C2C）
- 「気になる」ボタン：アクセントカラーで目立たせる

### タイポグラフィ方針
- 見出し：やや太め・読みやすいゴシック体（Noto Sans JP等）
- 本文：読みやすいサイズ（16px以上）、行間を広めに
- 雰囲気：雑誌的な余裕のある文字組み。ぎちぎちにしない

### アイコン・イラスト
- アイコン：シンプルなアウトラインアイコン（過度な装飾なし）
- イラスト：原則使わない。コンテンツ画像の質で勝負
- 「気になる」ボタン：ブックマークアイコン（押すとアースブラウンに変化）

## 3. UX原則

### 重視する体験

1. **「始めやすさ」** ── ジャンルを選ぶだけ、30秒で使い始められる。登録しなくても見られる。
2. **「眺める気持ちよさ」** ── 画像が大きく、余白があり、スクロールが気持ちいい。途中で止まらない。
3. **「貯まっていく充実感」** ── コレクションが増えるほど「自分の趣味マップ」が育っている感覚になる。

### 避けたいUX

1. **数字の氾濫** ── いいね数・フォロワー数・閲覧数を前面に出さない。「〇人が気になっています」のみに留める。
2. **通知の過多** ── バッジやプッシュ通知でユーザーを急かさない。使いたい時に使える設計。
3. **投稿への誘導** ── 「あなたも投稿しよう」的なCTAは絶対に出さない。

## 4. 画面構成

### 主要画面一覧
| 画面名 | 目的 | 主要要素 |
|--------|------|----------|
| ウェルカム画面 | コンセプト伝達・初回誘導 | キャッチコピー・「はじめる」ボタン |
| ジャンル選択画面 | フィード初期化 | ジャンルカード（キャンプ・温泉・古着等）、複数選択可 |
| ホームフィード | メイン体験 | カード型コンテンツ一覧、ジャンルタブ |
| カード詳細画面 | コンテンツ詳細確認 | 大画像・説明文・人数表示・「気になる」ボタン・外部リンク |
| コレクション画面 | 保存コンテンツの閲覧 | カテゴリタブ・サムネイル一覧 |
| 設定画面 | ジャンル変更・通知設定 | ジャンル設定・通知ON/OFF・アカウント情報 |

### 画面遷移図
```
[ウェルカム] → [ジャンル選択] → [ホームフィード]
                                        ↓
                                 [カード詳細] → [外部ブラウザ（X/note/YouTube）]
                                        ↓
                                 [コレクション保存]
                                        
[ホームフィード] ←→ [コレクション] ←→ [設定]
（ボトムナビゲーションで行き来）
```

## 5. インタラクション

### アニメーション方針
控えめに使用。スクロールはスムーズに、画面遷移はシンプルなフェード。
派手なアニメーションは使わない（落ち着いた体験を損なうため）。

### フィードバック
- 「気になる」タップ時：ブックマークアイコンが色変化（アースブラウンに）＋小さなバウンスアニメーション
- 保存完了：下部にトースト通知（「コレクションに保存しました」）
- 外部リンク遷移：アプリ内ブラウザが下からスライドインで表示

### エラー表示
- シンプルなメッセージ（「読み込めませんでした。もう一度お試しください」）
- エラー画面は最小限。すぐリトライできるボタンを用意
- オフライン時：「インターネット接続がありません。コレクションは引き続き見られます」
