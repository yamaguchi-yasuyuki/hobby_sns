# ユーザーフロー

## 1. オンボーディングフロー

### 初回起動
1. アプリを開く
2. 「投稿しなくていい。眺めるだけのSNS。」のコピーとともにウェルカム画面が表示される
3. 「はじめる」ボタンをタップ
4. 趣味ジャンル選択画面（キャンプ・温泉・サウナ・バイク・釣り・古着・ウイスキー・国内旅行 等から複数選択）
5. 「このジャンルで始める」→ ホームフィードへ遷移
6. （任意）アカウント作成の促し。「コレクションを保存するにはアカウントが必要です」

### アカウント登録（任意）
1. メールアドレスを入力
2. パスワードを設定
3. 登録完了 → コレクション機能が使えるようになる
※ アカウントなしでもフィードの閲覧は可能

---

## 2. メインフロー

### ホームフィードを眺める
**目的：** 趣味のビジュアル情報をインプットする

**フロー：**
1. アプリを開く → ホームフィードが表示される
2. 縦スクロールでカードを眺める
3. 気になる画像・タイトルのカードをタップ
4. カード詳細画面が開く（画像・説明文・出典情報・「〇人が気になっています」）
5. 「続きを読む」→ アプリ内ブラウザで外部コンテンツ（X・note・YouTube等）へ
6. 戻るボタンでフィードへ戻る

**成功条件：** ユーザーがスクロールを続け、1セッションで3枚以上のカードを閲覧する

---

### 「気になる」を保存する
**目的：** 後で見返すためにコレクションに追加する

**フロー：**
1. フィードまたはカード詳細で「気になる」ボタン（ブックマークアイコン）をタップ
2. コレクションカテゴリの選択（「行きたい場所」「欲しいもの」「気になるスポット」等）
3. 「保存しました」のトースト通知が表示される
4. コレクション画面でいつでも見返せる

**成功条件：** ユーザーが「気になる」を押した後、コレクション画面を確認しに行く

---

### コレクションを見返す
**目的：** 貯めた「気になる」を見返して次の行動を考える

**フロー：**
1. 下部ナビゲーションの「コレクション」タブをタップ
2. カテゴリ別にサムネイル一覧が表示される
3. 見たいカードをタップ → 詳細画面へ
4. 「続きを読む」→ 外部コンテンツへ or そのまま閉じる

**成功条件：** 保存したコンテンツから実際の行動（予約・購入等）につながる

---

## 3. その他のフロー

### ジャンルフィルター
1. ホームフィード上部のタブで「キャンプ」「温泉」「サウナ」「バイク」「釣り」「古着」「ウイスキー」「国内旅行」等を選択
2. 選んだジャンルのコンテンツだけに絞られる

### ジャンル設定の変更
1. 設定画面 → 「趣味ジャンルを変更する」
2. ジャンル選択画面（初回オンボーディングと同じ画面）
3. 変更後はフィードが更新される

### ログイン（2回目以降）
1. アプリを開く → 自動ログイン（セッション保持）
2. セッション切れの場合：メールアドレス＋パスワードでログイン
