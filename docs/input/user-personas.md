# ターゲットユーザー

## メインペルソナ

### 基本情報
- 名前（仮）：山田 浩二（やまだ こうじ）
- 年齢：44歳
- 職業：会社員（中間管理職）
- 生活環境：妻・子供と郊外在住。週末は趣味の時間を確保している
- 趣味：キャンプ・温泉・バイク（ツーリング）・ウイスキー

### 1日の流れ
- 07:00 出勤。通勤電車でYouTubeやニュースをチェック
- 09:00〜18:00 仕事
- 19:00 帰宅。夕食・家族との時間
- 21:00 自分の時間。趣味の道具を眺めたり、次のキャンプ計画を考える
- 23:00 就寝前にスマホでYouTubeを流し見

### 課題・ペイン
- キャンプや温泉の情報を探したいが、Googleで検索するのが毎回手間
- YouTubeはいいが、「ちょうどいい情報が流れてくる場所」がない
- Instagramは参考になりそうだが、アカウントを作ること・投稿することへの抵抗感がある
- SNSは「若い人がやるもの」「自己顕示欲の強い人がやるもの」というイメージがある

### 現在の解決方法
- Google検索で都度調べる
- YouTubeで好きなキャンパーの動画を見る
- 雑誌（BE-PAL、Fielder等）を読む
- 友人からの口コミ

### 求めているもの
- 手間なく、趣味の良質なビジュアル情報が流れてくる場所
- 投稿したり、フォローしたりという「SNS的な義務」がない体験
- 気になったものを後で見返せる仕組み
- 同世代が何を良いと思っているかの参考情報

### このプロダクトを使う瞬間
- 通勤電車の中でぼーっと眺めたい時
- 就寝前のリラックスタイムに次のキャンプ・ツーリング計画を妄想する時
- 週末の計画を立てる時（行き先・欲しい道具・温泉スポットを探す）
- 家でPCを開いてゆっくり情報収集する時

### 情報収集の方法
- Google検索（能動的な調べもの）
- YouTube（動画で学ぶ）
- 友人・同僚からの口コミ
- 書店でたまたま見かけた雑誌

### 課題・ペイン

- 「いいねを押す」「フォローする」という行為自体が面倒くさい

- 趣味ごとにアプリを使い分けるのが面倒

### 求めているもの
- 複数の趣味をまとめて眺められる場所
- 「いいね数」「フォロワー数」という数字が見えない、静かなフィード
- 欲しいもの・行きたい場所を自分用に保存できる仕組み

## 非ターゲット
- 積極的に発信したい人（Instagramで十分）
- フォロワーを増やしたい人（Twitterで十分）
- 承認欲求を満たしたい人（既存SNS各種で十分）
- 10〜20代のデジタルネイティブ（SNSへの抵抗感がそもそも少ない）
