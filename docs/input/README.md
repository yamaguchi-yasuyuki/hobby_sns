# MITORI（読むだけSNS）

## 概要
40代男性の「発信はしたくないけど、趣味の情報は欲しい」というニーズに応える、完全受信型のビジュアルフィードアプリ。投稿機能を持たず、キャンプ・温泉・バイク・釣り・古着・ウイスキーなど40代男性のライフスタイルコンテンツを眺めて、気になったものをコレクションに貯めていく。

## 解決する課題
SNSをやっていない40代男性の多くは「発信したいことがない」「必要性を感じない」という理由でSNSから離れている。しかし趣味の良質な情報は欲しい。既存のSNSは「発信しなければいけない」という暗黙のプレッシャーがあり、参加の入口が高い。

## ターゲットユーザー
40代男性・SNS未使用・プライベートの充実に趣味を持つ人（キャンプ・温泉・サウナ・バイク・釣り・古着・ウイスキー・国内旅行 等）

## 主要機能
- ビジュアル中心のホームフィード（投稿機能なし）
- 「気になる」ボタンで個人コレクションに保存
- コレクション一覧（カテゴリ別：行きたい場所・欲しいもの等）
- 外部コンテンツ（X・note・YouTube）へのシームレスな導線
- 「〇人が気になっています」の人数表示のみの社会性
